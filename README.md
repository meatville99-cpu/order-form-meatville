# Meatville Order Form App (PWA)

One-page A4 print-accurate web app. Works on tablet + PC and can be installed to home screen.

## Run locally
1) Install Node.js 18+
2) In this folder:
   npm install
   npm run dev

## Build / deploy
npm run build
Deploy the `dist/` folder to any static host (Netlify/Vercel/Cloudflare Pages).

## Install
- iPad/iPhone Safari: Share -> Add to Home Screen
- Android Chrome: Menu -> Install app
- Desktop Chrome/Edge: Install icon in address bar

## Printing
Print on A4, scale 100%.
